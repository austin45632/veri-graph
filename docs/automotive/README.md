# Automotive Feature Samples

這個目錄放車載儀表系統 knowledge graph 的第一組車載語意樣例，用來驗證雙主軸模型不是只有 schema。

v1 樣例節點：
- `FEAT-001`: 車速顯示功能
- `VSTATE-001`: 點火 ON 且車輛行駛中
- `SIG-001`: CAN 車速訊號
- `FAULT-001`: 車速訊號逾時後的降級顯示反應
- `DIAG-001`: 車速訊號逾時診斷事件
- `VAR-001`: 車型高配變體
- `SG-001`: 駕駛不得被誤導的顯示安全目標

這些節點透過關聯接到既有 `REQ-001`、`TR-001`、`TI-001`、`TC-001`，但不改變最低追溯合規 gate。
